Install PlanSync

$ npm install -g plansync

Prerequisites

  • Node.js 18+ — PlanSync runs on any LTS version of Node.js
  • A GitHub account — with a repository you want to manage

Authenticate

Run plansync init in your project and paste a GitHub Personal Access Token when prompted:

$ cd my-project
$ plansync init

Create a token at github.com/settings/tokens/new with repo, issues, and project scopes. Paste it into the prompt — PlanSync validates it and saves it to .plansync/config.json.

No token handy? Just press Enter and PlanSync falls back to browser-based device flow. A built-in Client ID is included — no setup needed.

Common errors

ErrorFix
No git remote "origin"Push your repo to GitHub first
Invalid tokenGenerate a new token at github.com/settings/tokens/new with repo + issues + project scopes
Permission deniedUse sudo npm install -g or fix your npm permissions
