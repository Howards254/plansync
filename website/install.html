Install PlanSync

npm install -g plansync

Prerequisites

  • Node.js 18+ — PlanSync runs on any LTS version of Node.js
  • A GitHub account — with a repository you want to manage
  • An Anthropic API key — for LLM-powered plan generation (get one here)
  • A GitHub OAuth App — create one at github.com/settings/developers (no callback URL needed for device flow)

Set environment variables

export ANTHROPIC_API_KEY=sk-ant-...
export PLANSYNC_GITHUB_CLIENT_ID=your_github_oauth_client_id

Initialize

Navigate to your GitHub repository and run:

cd my-project
plansync init

Follow the device-code authentication flow. PlanSync will save your credentials locally and scaffold the project.

Common install errors

Permission deniedUse sudo npm install -g or fix your npm permissions
